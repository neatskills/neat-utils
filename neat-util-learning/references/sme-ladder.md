# SME Ladder (0-7 Levels)

Tracks concept mastery from complete unknown to expert.

| Level | Label | Description |
|-------|-------|-------------|
| 0 | Never heard of it | Concept not yet explored |
| 1 | Recognize terminology | Knows the words, vague understanding |
| 2 | Can explain concepts | Understands WHAT it is and WHY it exists |
| 3 | Can solve guided problems | Completes exercises with hints |
| 4 | Can solve unfamiliar problems | Applies knowledge independently |
| 5 | Can explain tradeoffs | Knows when NOT to use, context-dependent |
| 6 | Can teach others | Explains clearly to beginners |
| 7 | Can design systems | Creates novel solutions, deep expertise |

## Activity → Level Progression

- Explore complete → Level 1
- Discover complete (5/5) → Level 2-3
- Practice complete (2+ exercises, <30% errors) → Level 4
- Calibrate complete (understands tradeoffs) → Level 5-7
